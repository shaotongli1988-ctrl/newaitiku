# 刷题升本三子模块与闯关积分验收说明（2026-03-23）

## Given / When / Then

### 1. 旧入口兼容
- Given 学生从旧入口 `/student/practice` 进入
- When 未传模块参数，或历史参数为 `tab=chapter`
- Then 页面跳转到章节闯关子路由，并保留原有 query 语义

### 2. 章节闯关
- Given 学生进入章节闯关
- When 选择章节或 L5 知识点
- Then 页面仅保留紧凑工作条、答题区与必要导航，不再显示冗余工作台卡片

### 3. 自由练习
- Given 学生进入自由练习
- When 按科目或知识点开始练习
- Then 题目来源不受章节解锁限制，页面默认只保留答题主区与积分反馈

### 4. 模拟考试
- Given 学生进入模拟考试
- When 选择推荐试卷或手动切换试卷
- Then 页面展示试卷答题区，客观题可即时记分，主观题不计即时闯关积分

### 5. 闯关积分
- Given 学生在章节闯关、自由练习或模拟考试中答题
- When 同一题目第一次答对
- Then 增加 1 个闯关积分；后续再次答对同题不重复加分

### 6. 排行与段位
- Given 学生有当前科目练习积分
- When 进入学习首页或练习积分页
- Then 可查看全站同科目排名、今日新增积分、按 10000 分总分制计算的当前段位、下一段位差距与分享入口
- Then 练习积分页当前段位旁必须展示对应段位形象图，青铜/白银/黄金/铂金/钻石/星耀/王者均有独立视觉
- Then 下载海报区域旁必须提供可扫码二维码，扫码后可进入手机端当前积分页进行打卡助力与分享

### 7. 打卡与任务奖励
- Given 学生完成每日打卡或任务奖励条件
- When 奖励发放
- Then 通用积分奖励统一按 2 分发放，不影响独立闯关积分账本

## UAT 重点
- UAT-1：`Home -> Practice(Chapter/Free/Mock) -> Points` 主链路可达
- UAT-2：章节闯关仍执行解锁规则，自由练习不受章节锁限制
- UAT-3：章节闯关、自由练习、模拟考试三条线都能触发闯关积分
- UAT-4：同题重复答对不重复记分
- UAT-5：首页“今日主线”和积分页数据一致
- UAT-6：练习积分页二维码可扫到手机端，未打卡时可完成今日打卡并刷新积分状态
