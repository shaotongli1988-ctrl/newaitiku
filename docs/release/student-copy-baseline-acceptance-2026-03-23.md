# 学生端页面文案基线验收清单（2026-03-23）

## 1. 目的

- 把本轮学生端真点击回放里实际使用到的页面文案锚点沉淀为稳定验收口径。
- 避免后续页面改版时出现“功能能点通，但文案锚点和自动化 / UAT 口径脱节”。
- 当前口径优先服务学生端核心导航链路：`首页 -> 知识诊断（诊断总览 / 今日任务 / 练习积分） -> 刷题升本 -> 我的题库`。

## 2. 适用范围

- 学生端桌面端核心导航真点击回放
- 学生端手工回归与 UAT
- 学生端页面文案改版后的基线复核

## 3. 页面文案基线

| 页面 | 路由 | 必须可见的文案锚点 | 说明 |
| --- | --- | --- | --- |
| 学习首页 | `/student/home` | `今日主线`、`刷题段位已经提到首页主位`、`章节闯关`、`自由练习`、`模拟考试` | 首页首屏必须可读出当前学习主线与练习入口 |
| 知识诊断总览 | `/student/analysis/overview` | `知识诊断`、`诊断总览`、`今日任务`、`练习积分`、`知识结构与薄弱点诊断`、`当前科目` | 一级导航进入后默认落总览子页 |

| 知识诊断练习积分 | `/student/analysis/points` | `知识诊断`、`诊断总览`、`今日任务`、`练习积分` | 练习积分现为知识诊断二级详情页 |

| 我的题库错题中心 | `/student/question-bank/repair` | `AI 修复中心`、`开始修复` | 旧长文案不再作为稳定验收锚点 |

## 4. Given / When / Then 验收条目

### 4.1 首页文案基线

- Given 学生账号成功登录并进入学习首页
- When 首页首屏加载完成
- Then 必须能看到 `今日主线`
- Then 必须能看到 `刷题段位已经提到首页主位`
- Then 必须能看到 `章节闯关`、`自由练习`、`模拟考试`
- Then 不再以 `闯关积分` 作为首页主锚点

### 4.2 知识诊断导航基线

- Given 学生位于任一学生端主页面
- When 点击一级导航 `知识诊断`
- Then 路由应进入 `/student/analysis/overview`
- Then 页面必须能看到 `知识结构与薄弱点诊断`
- Then 页面必须能看到 `当前科目`
- Then 页头必须可见二级页签 `诊断总览`、`今日任务` 与 `练习积分`

### 4.3 知识诊断今日任务基线

- Given 学生已进入知识诊断模块
- When 切换到二级页 `今日任务`
- Then 路由应进入 `/student/analysis/tasks`
- Then 页面标题应使用 `知识诊断今日任务`
- Then 页面语义应表达“诊断之后的执行动作”，不应再退回旧的独立任务中心命名
- Then 页面必须明确说明今日任务与执行记录按全局汇总展示
- Then 页面必须明确说明弱项提醒覆盖全部科目

### 4.3A 知识诊断今日任务切科目基线

- Given 学生位于 `知识诊断 > 今日任务`
- When 切换顶部当前科目
- Then 页面中关于 `全局学习节奏` 与 `不按当前科目拆分` 的提示应保持不变
- Then 页面中关于 `弱项提醒覆盖全部科目` 的提示应保持不变
- Then 页面不得出现“弱项仅看当前科目”的误导性表达

### 4.4 刷题升本文案基线

- Given 学生点击一级导航 `刷题升本`
- When 进入章节练习页
- Then 页面必须能看到 `刷题升本`
- Then 页面必须能看到 `重置路径`
- Then 页面必须能看到 `章节闯关`、`自由练习`、`模拟考试`
- Then 页面必须能看到 `选择 L3 模块`、`选择 L4 章节`、`选择 L5 考点`
- Then 不再要求出现旧的 `L3 / L4 / L5` 按钮文本

### 4.5 知识诊断练习积分文案基线

- Given 学生已进入知识诊断模块
- When 切换到二级页 `练习积分`
- Then 路由应进入 `/student/analysis/points`
- Then 页面应以 `练习积分` 为主要文案锚点
- Then 不再以 `闯关积分` 作为验收锚点

### 4.6 我的题库错题中心文案基线

- Given 学生点击一级导航 `我的题库`
- When 页面落到 `/student/question-bank/repair`
- Then 页面必须能看到 `AI 修复中心`
- Then 页面必须能看到 `开始修复`
- Then 不再要求出现旧的长段引导文案作为稳定锚点

## 5. UAT 执行步骤

1. 用学生种子账号登录：`13800000005 / seed-password-student-001`
2. 检查学习首页首屏是否满足首页文案基线
3. 点击 `知识诊断`，检查是否默认进入 `诊断总览`
4. 在知识诊断中确认二级页签 `诊断总览 / 今日任务 / 练习积分` 可见
5. 切到 `今日任务` 后，确认页面出现 `全局学习节奏`、`不按当前科目拆分`、`弱项提醒覆盖全部科目`
6. 切到 `练习积分` 二级页，确认页面使用 `练习积分` 文案
7. 点击一级导航 `刷题升本`，检查路径重置入口与三层选择器锚点
8. 点击一级导航 `我的题库`，确认错题中心首屏使用 `AI 修复中心 / 开始修复`
