# 老师双岗权限回滚与止血开关（2026-03-28）

## kill switch / feature switch

- switch 名称：`teacher_post_tags_switch`
- 作用：快速将双岗联动能力回切为单岗策略，止血权限异常影响。
- 生效方式：超管端优先回切岗位标签，再触发服务配置刷新。

## 回滚策略

- 应用回滚：回退到上一稳定版本。
- 配置回滚：恢复上一版权限配置快照。
- 数据库影响：本次不涉及 schema 变更，无数据回填回滚。

## 触发条件

- 出现批量越权或权限错配。
- 教师首页核心模块大面积不可见或误可见。

## 验证步骤

- 回切后 5 分钟内复测：招生岗、教学岗、双岗、超管四类账号。
- 预期：权限显隐恢复到回滚前稳定状态。
