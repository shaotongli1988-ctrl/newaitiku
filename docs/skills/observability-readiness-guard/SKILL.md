# 可观测性就绪

## 核心目标

- 确保上线后问题可发现、可定位、可追踪。

## 触发场景

- 发布前检查
- 监控建设
- 重大改造上线

## 默认检查项

1. 关键日志字段是否齐全
2. trace / metrics / 埋点是否覆盖关键链路
3. 告警是否建立

## 输出要求

- 输出可观测性覆盖清单
- 输出缺失监控项
