# 应用安全基线

## 核心目标

- 把安全检查变成上线前必须过的基线门禁。

## 触发场景

- 发布前安全检查
- 重要接口改造
- 权限与鉴权改造

## 默认检查项

1. 鉴权与权限控制
2. 输入校验与输出转义
3. SQL 注入、XSS、CSRF 风险
4. 防重复提交与幂等
5. 敏感信息是否出现在代码或日志

## 输出要求

- 输出安全风险分级
- 指出阻断项与修复建议
